## Initialize Magnific Popup Plugin


<pre><code>
    &lt;div class="popup-container"&gt;
        &lt;a href="images_path_here"&gt;
            &lt;img src="images_path_here" alt="" /&gt;

    &lt;/div&gt;
</code></pre>


<pre><code>
     $('.popup-container').magnificPopup({<br />
        delegate: 'a',\
        type:'image',\
        gallery:{enabled:true},\
        zoom:{\
            enabled:true,\
            duration: 300,\
            easing: 'ease-in-out'\
        }\
    });
</code></pre>
